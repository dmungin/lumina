# Description

A Rust monorepository that holds the Lumina (a self hosted communication platform) source code.

## PreReqs

- [Rust](https://rust-lang.org/tools/install/)
- [Tauri](https://v2.tauri.app/)

*Some Container Technology*

## Creating The Applications Locally

```sh
cargo run -p <cargo toml member name>
```

## Building The Container Swarm

```sh
<container tech> compose up -d
```
